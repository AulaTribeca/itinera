# ITINERA v0.25 · versión simplificada y simulador mejorado

## Cambios

- Se deja solo lo esencial: portada, buscador sencillo, simulador de itinerarios, ItineraBot, selector de idioma, selector de fuente y menú desplegable.
- El menú desplegable organiza herramientas y enlaces oficiales.
- Se ocultan secciones redundantes: itinerarios, bachillerato, FP, universidad, becas, cupos, convalidaciones, pruebas, FAQ y paneles de estado.
- El pie de página no se modifica.
- El simulador tiene estética más limpia y elimina sombras alargadas.
- El simulador indica Bachillerato recomendado cuando procede.
- El simulador contempla rutas a ciclo superior sin pasar necesariamente por ciclo medio o Bachillerato, mediante prueba de acceso si se cumplen requisitos.
- Ajuste de ruta queda reducido a dos opciones: ruta más corta o mostrar todas las rutas.
